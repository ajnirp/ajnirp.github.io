You have a homogenous "values" array, and a "weights" array of equal length containing floats. The weights array tells you the relative weight of each element in the values array. Write a function to sample from the values array that respects the weights. For example, if the weights are `[1.4, 4.2]`, then `values[0]` should be chosen 25% of the time, and `values[1]` should be chosen 75% of the time.
